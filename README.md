# 책첵 BookCheck
## 💻 프로젝트 개요
- 주제
  - 독서 동아리 활동을 통한 도서 나눔과 월 별 모임 기록, 월 별 모임 후기를 공유하는 서비스 제공
- 개발목적
  - 정기적인 모임을 통해 다양한 도서 정보 공유
- 대상
  - 서울에 거주하는 책을 사랑하는 성인
- 주요기능
  - 도서 나눔, 모임 이력, 모임 후기, 도서 트렌드 분석
## ✋ 팀 소개
- 서정욱
  - 팀장 / 백엔드
  - REST API, DB테이블 구성, 사용자 페이지 모임 기록 및 후기, 도선 남 등록 및 신청
- 김민수
- 송하은
- 원가희
